# Bright Motors Car Sales Analysis
# 1. Problem Statement
Bright Motors has recently appointed a new Head of Sales with a mandate to expand dealership operations, improve overall sales performance and optimize inventory management. However, decision-making is currently limited due to a lack of clear insights from historical sales data.
The challenge is to analyze the car sales dataset and extract meaningful insights that will support data-driven decisions in sales strategy, customer targeting and operational efficiency.
# 2. Aim of the Project
The aim of this project is to leverage data analytics techniques to uncover trends, patterns and key performance indicators within the car sales dataset.
This analysis is intended to provide actionable insights that will help Bright Motors:
  1. Increase revenue
  2. Improve sales performance
  3. Optimize inventory and pricing strategies
  4. Understand customer preferences
# 3. Objectives (Steps Taken to Achieve the Aim)
  ## Data Preparation
Converted raw Excel dataset into CSV format
Uploaded data into SQL environment (e.g., Snowflake / Databricks / SQL Server)
Cleaned data by:
  1. Removing duplicates
  2. Handling missing values
  3. Converting price fields into numeric format
  ## Data Transformation
Created calculated fields such as:
1. Total Revenue = Selling Price × Units Sold
2. Profit Margin = (Selling Price – Cost Price) ÷ Selling Price × 100
3. Categorized vehicles into performance tiers (High, Medium, Low margin)
Grouped data by dimensions such as:
1. Car make and model
2. Year of manufacture
3. Region
4. Fuel type
  ## Data Analysis
Analyzed:
1. Top-performing car makes and models by revenue
2. Relationship between price, mileage, and vehicle age
3. Regional sales performance
4. Customer purchasing trends over time
  ## Data Visualization
Built interactive dashboards using tools like Power BI / Excel / Looker Studio
Included filters (slicers) for:
1. Region
2. Fuel type
3. Year
  ## Presentation
Compiled insights into a business presentation
Provided recommendations for improving sales and profitability
# 4. Summary of Results
1. Identified top-performing car brands and models contributing the highest revenue
2. Observed pricing trends influenced by mileage and year of manufacture
3. Highlighted regions with the highest and lowest sales performance
4. Discovered customer preferences shifting towards specific fuel types and vehicle categories
5. Provided recommendations to:
6. Focus on high-margin vehicles
7. Optimize pricing strategies
8. Expand in high-performing regions
9. Improve inventory planning
